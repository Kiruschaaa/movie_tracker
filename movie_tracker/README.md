# 🎬 Movie Tracker — Трекер фильмов

Консольное приложение для учёта фильмов. Позволяет добавлять, просматривать, редактировать и удалять фильмы, а также фильтровать их по жанру или году.

## Запуск

```bash
python movies.py
```

Требования: Python 3.8+, стандартная библиотека (sqlite3 встроен).

## Запуск тестов

```bash
pip install pytest pytest-cov
pytest tests/ -v
pytest tests/ --cov=movies --cov-report=term-missing
```

## Архитектура

```
пользователь
    │
    ▼
  menu()              ← консольный интерфейс
    │
    ▼
бизнес-логика:
  add_movie()
  get_all_movies()
  get_movie_by_id()
  update_movie()
  delete_movie()
  filter_by_genre()
  filter_by_year()
    │
    ▼
  SQLite (movies.db)  ← хранение данных
```

## Структура данных

Таблица `movies`:

| Поле    | Тип     | Описание                    |
|---------|---------|-----------------------------|
| id      | INTEGER | Автоинкремент, первичный ключ |
| title   | TEXT    | Название фильма             |
| year    | INTEGER | Год выхода (1888–2100)      |
| rating  | REAL    | Рейтинг (0.0–10.0)          |
| genre   | TEXT    | Жанр                        |

## Функциональность (MVP)

- Добавление фильма с валидацией полей
- Просмотр всех фильмов
- Редактирование записи по ID
- Удаление записи по ID
- Фильтрация по жанру (без учёта регистра)
- Фильтрация по году

## Версии

- **1.0.2** — фикс по баг-репорту: рейтинг 0.0 принимается корректно
